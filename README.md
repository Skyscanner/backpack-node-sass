# backpack-node-sass

[![Build Status](https://travis-ci.org/Skyscanner/backpack-node-sass.svg?branch=master)](https://travis-ci.org/Skyscanner/backpack-node-sass) 

Parallelised `node-sass` with Backpack support.

## Install

```
npm install --save-dev backpack-node-sass
```

## Usage

```
> backpack-node-sass
✔ 159 files found
✔ 8 workers spawned
✔ 159/159 compiled
```

Watch mode:

```
> backpack-node-sass --watch
✔ Ready for changes
✔ Compiled: packages/bpk-component-button/src/bpk-button.css
✔ Compiled: packages/bpk-component-text/src/BpkText.css
```
